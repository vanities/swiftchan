## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Download [xCode](https://apps.apple.com/us/app/xcode/id497799835?mt=12)

### Installation

1. Clone the repo
```sh
git clone https://github.com/vanities/swiftchan
```
2. Install pod pakages
```sh
pod install
```
